**Subject: Information request — Catharina van Clevepark 40 (Funda 44469451)**

Dear Sir/Madam,

Thank you for the viewing of Catharina van Clevepark 40. We are interested in the property. Before submitting a well-founded offer, we would like to assess the apartment and the owners' association (VvE) carefully. Could you send us the following documents?

**Legal / ownership**

- Proof of ownership (deed of transfer) and Land Registry extract

- Division regulations and house rules

- Building insurance policy
- Confirmation the VvE is active

**Structural / condition (built 1951)**

- NEN2580 measurement report (to resolve the floor-area discrepancy: Funda 85 m² vs. Kadaster 98 m²)
- Any structural survey report
- Asbestos inventory, if available
- Definitive energy label (EP-Online registration)

**Protected townscape / permits**

- Confirmation the existing plastic window frames were permitted (omgevingsvergunning)
- Permits for any earlier changes to the facade/roof

**Pests**

- Any history and/or reports regarding mice/rats in the unit or the building
- Any ongoing pest-control contracts or measures within the VvE
- Information on the building's waste container (management/placement)

We would appreciate receiving these so we can submit an offer shortly. Thank you in advance.
